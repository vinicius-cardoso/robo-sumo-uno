# Firmware

Codigo embarcado do robo.

- `arduino/robo_sumo_uno/`: sketch principal do Arduino.
- `libraries/`: bibliotecas locais do projeto, se houver.
- `tests/`: testes, sketches auxiliares e validacoes de sensores/atuadores.

No Arduino IDE, o arquivo `.ino` principal deve ter o mesmo nome da pasta do
sketch. Para este projeto, use:

```text
firmware/arduino/robo_sumo_uno/robo_sumo_uno.ino
```

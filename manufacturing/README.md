# Manufacturing

Pasta para arquivos e instrucoes finais de fabricacao e montagem.

- `pcb/`: instrucoes para pedido e montagem da PCB.
- `3d-print/`: parametros de impressao, orientacao e material.
- `laser-cut/`: espessura, material, kerf e fornecedor.
- `steel/`: material, acabamento, tolerancias e processo.

Use esta pasta para o pacote final enviado a terceiros ou usado na bancada.

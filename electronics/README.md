# Electronics

Arquivos de eletronica do robo.

- `kicad/`: projeto principal do KiCad.
- `libraries/`: simbolos, footprints e modelos 3D especificos do projeto.
- `datasheets/`: datasheets usados para escolha e validacao de componentes.
- `pinouts/`: imagens e referencias rapidas de pinagem dos modulos.
- `manufacturing/`: Gerbers, arquivos de montagem, pick-and-place e relatorios.

Mantenha bibliotecas customizadas com caminhos relativos ao repositorio sempre
que possivel, para o projeto abrir corretamente em outra maquina.

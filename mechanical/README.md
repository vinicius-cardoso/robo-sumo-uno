# Mechanical

Arquivos mecanicos do robo.

- `cad/assemblies/`: montagens completas.
- `cad/printed-3d/`: pecas modeladas para impressao 3D.
- `cad/laser-cut-acrylic/`: pecas para corte a laser em acrilico.
- `cad/steel-parts/`: pecas para fabricacao em aco.
- `exports/stl/`: STLs para impressao 3D.
- `exports/step/`: STEP para troca entre CADs e fornecedores.
- `exports/dxf/`: DXF para corte a laser, plasma, jato d'agua ou CNC.
- `drawings/`: desenhos tecnicos em PDF ou formato CAD.

Procure manter uma relacao clara entre o arquivo editavel e o exportado. Exemplo:
`cad/laser-cut-acrylic/front_wedge_v03.*` e
`exports/dxf/front_wedge_v03.dxf`.

# Media

Fotos, renders e videos do projeto.

- `photos/`: fotos de montagem, testes e revisoes.
- `renders/`: imagens renderizadas das pecas ou montagens.
- `videos/`: videos curtos de teste ou demonstracao.

Evite versionar videos muito grandes diretamente no Git. Prefira Git LFS ou
links externos quando o arquivo crescer demais.
